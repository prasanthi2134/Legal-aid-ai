# ⚖️ Legal Aid AI

An AI-powered legal guidance chatbot for Indian citizens.
Helps users understand their legal rights in English and Hindi.

🔗 Live Demo: prasanthi2134.github.io/legal-aid-ai

---

## 🤖 How This Was Built

This project was built entirely using AI prompting tools as part of
Hack2Skills x Google Developers — Prompt Wars.

No code was written manually. Everything was generated and refined
by writing smart structured prompts across multiple AI tools.

### AI Tools Used

| Tool | How I Used It |
|---|---|
| Claude by Anthropic | Main code generation — HTML, CSS, JavaScript, knowledge base, voice input, language toggle |
| ChatGPT by OpenAI | Brainstorming legal categories, Hindi translations, improving response structure |
| Antigravity | Running, testing and previewing the code live in the browser |

---

## 💡 Prompts I Used

- Build a legal aid chatbot UI that looks like ChatGPT with a dark sidebar
- Add a knowledge base with FIR, cybercrime, property dispute, RTI categories with step by step guidance
- Add official Indian government links as clickable buttons inside each chatbot response
- Add voice input using Web Speech API with a microphone button
- Add English and Hindi language toggle that changes all chatbot responses
- Fix the SyntaxError — currentLang declared twice — without changing functionality
- Make the UI more modern and professional like ChatGPT
- Add more quick action buttons to the sidebar

---

## ✨ Features

- 15 plus legal categories with step by step guidance
- Official government links in every answer
- Voice input using Web Speech API
- English and Hindi language toggle

- 9 quick action sidebar buttons
